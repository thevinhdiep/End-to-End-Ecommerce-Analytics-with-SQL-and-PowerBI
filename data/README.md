# 📂 Kho Dữ liệu (Data)

Thư mục này chứa toàn bộ dữ liệu (Raw & Cleaned) được sử dụng trong suốt dự án E-Commerce Analytics.

## 🗂 Cấu trúc thư mục

- `raw/`: Chứa file dữ liệu gốc tải về từ nguồn. Dữ liệu này ở dạng nguyên bản, chưa qua bất kỳ thao tác xử lý nào.
- `cleaned/`: Chứa các file dữ liệu đã được làm sạch và biến đổi thông qua Jupyter Notebook (đầu ra của Phase 2). File chính là `cleaned_ecommerce_dataset.csv`. Dữ liệu trong này đã hoàn toàn sạch và sẵn sàng để nạp vào Data Warehouse.
