# 📊 Báo cáo Kiểm toán (Audit Reports)

Thư mục này lưu trữ các tài liệu tổng kết sau quá trình dùng Python phân tích chất lượng dữ liệu ở Phase 1.

## 🗂 Danh sách Báo cáo

- **`data_audit_summary.md`**: File tóm tắt các vấn đề về chất lượng dữ liệu phát hiện được, bao gồm số lượng missing values, đánh giá tính logic của các metrics và các khuyến nghị xử lý trước khi thực hiện ETL.
- **`data_audit_tables/`**: Chứa các bảng số liệu phụ trợ xuất ra từ quá trình Audit để dễ dàng đối chiếu.
