# 📓 Jupyter Notebooks (Xử lý & Khám phá Dữ liệu)

Thư mục này chứa mã nguồn Python (Pandas, Numpy) cho quá trình Kiểm toán (Data Audit) và Làm sạch dữ liệu (Data Cleaning).

## 🗂 Danh sách Notebook

1. **`01_data_audit.ipynb`**: Kịch bản đọc file CSV thô, kiểm tra missing values, duplicates, outliers và phân tích phân phối dữ liệu (Phase 1).
2. **`02_data_cleaning.ipynb`**: Kịch bản xử lý lỗi, chuẩn hóa tên cột, tạo Surrogate Key (`order_id`) và xuất ra dữ liệu sạch (Phase 2).
