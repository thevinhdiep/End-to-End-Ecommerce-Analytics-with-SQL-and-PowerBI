# 📚 Tài liệu Hệ thống (Documentation)

Thư mục này chứa các tài liệu quy chuẩn, từ điển dữ liệu và sơ đồ kiến trúc của hệ thống.

## 🗂 Danh sách Tài liệu

1. **`data_dictionary.md`**: Giải thích chi tiết ý nghĩa nghiệp vụ của từng cột trong file dữ liệu.
2. **`data_model.md`**: Mô tả sơ đồ thực thể liên kết (ERD) và chiến lược thiết kế Star Schema cho Data Warehouse.
3. **`github_upload_guide.md`**: Cẩm nang và quy tắc thao tác upload file lên kho lưu trữ GitHub của dự án.
